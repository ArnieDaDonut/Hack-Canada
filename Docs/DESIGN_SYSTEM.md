# Design System 🎨

Estator AI follows a modern, premium design aesthetic characterized by "Clean Tech" and "Glassmorphism".

## Visual Principles

### 1. Glassmorphism
We use semi-transparent overlays with heavy blurs to create depth and a modern "premium" feel.
- **Background**: `rgba(255, 255, 255, 0.7)` (Light) or `rgba(15, 17, 26, 0.8)` (Dark).
- **Blur**: `backdrop-filter: blur(12px)`.
- **Border**: Subtle `1px solid rgba(255,255,255,0.1)`.

### 2. Color Palette
- **Primary**: `#3b82f6` (Vibrant Blue) - Used for primary actions and highlights.
- **Success**: `#10b981` (Emerald Green) - Used for confirmation and "Enhanced" states.
- **Dark Mode**: `#0f111a` (Deep Navy) - The primary background for high-contrast sections.

### 3. Typography
- **Headings**: `Outfit` or `Inter` (Sans-serif) for a bold, reliable tech aesthetic.
- **Body**: `Inter` for maximum readability across various screen sizes.

### 4. Iconography
- **Lucide React**: Used for functional UI elements (Upload, Layers, Filters).
- **Material Symbols**: Used for specific real-estate categories (Kitchen, Bed, Bath).
